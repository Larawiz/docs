# Soft Deletes

To make a model soft-deletable (opens new window), just issue the softDeletes or softDeletesTz into the columns list. Larawiz will automatically detect and use the SoftDeletes trait for the Model.




    Â 

    models:
  Post:
    title: string
    softDeletes: ~


    Â 




    class Post extends Model
{
    use SoftDeletes;
    
    // ...
}



    Â 



    Schema::create('posts', function (Blueprint $table) {
    $table->id();
    $table->string('title');
    $table->softDeletes();
    $table->timestamps();
});

    Alternatively, you can issue the column name to use as soft-deletes, that will be reflected in the model itself.




      Â 

      models:
  Post:
    title: string
    softDeletes: soft_deleted_at

      class Podcast extends Model
{
    use SoftDeletes;

    protected const DELETED_AT = 'soft_deleted_at';
}



      Â 



      Schema::create('posts', function (Blueprint $table) {
    $table->id();
    $table->string('title');
    $table->softDeletes('soft_deleted_at');
    $table->timestamps();
});

      Soft-deletes are always timestamps

      Laravel doesn't support non-timestamp soft-delete columns, but you're free to create your own soft-deleted column logic after scaffolding.
